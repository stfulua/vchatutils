# vChatUtils by vProLabs

Advanced chat management plugin for Paper 1.21.x servers with multi-language support.

![vChatUtils Banner](https://cdn.modrinth.com/data/cached_images/fee0ee72be8e5bc7b991f1a7d781ffa77814ab78_0.webp)

> If you find any bugs, report them on our [discord](https://discord.gg/SNzUYWbc5Q)!

## ✨ Full Features

**Chat Management**
- Disable/enable chat globally
- Clear chat with broadcast
- Hide join/leave messages
- Hide advancement announcements
- Bypass permission (When chat is disabled)

**Multi-Language**
- English (default) & Polish built-in
- Easy to add more languages
- All messages editable

---

## 🚀 Commands

| Command | Aliases | Description |
|---------|---------|-------------|
| `/chat clear` | `/chat wyczysc` | Clears chat |
| `/chat on` | `/chat wlacz` | Enables chat |
| `/chat off` | `/chat wylacz` | Disables chat |
| `/chat status` | `/chat info` | Shows settings |
| `/chat reload` | - | Reloads config |

---

## 🔒 Permissions

- `vchatutils.admin` - All commands (default: op)
- `vchatutils.bypass` - Write when disabled (default: op)

---

## 📝 Configuration

All settings in `config.yml` and `lang/` folder. Check files for detailed options.
